1. Change Detection Strategies:

  • Implement a parent-child component relationship where both components use different change detection strategies (Default and OnPush).
  • The parent component should have a button to update its own primitive and non-primitive data.
  • The child component should display the data passed from the parent and have a button to update its own data.
  • Demonstrate how the change detection strategy affects when and how the components are updated.
  • Showcase the behaviour of all possible scenario of Angular's OnPush change detection strategy when updating component data
